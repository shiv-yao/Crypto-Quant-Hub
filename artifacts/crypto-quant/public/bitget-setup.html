Bitget API 設定與測試

Bitget 憑證只放在 Railway Variables。此頁不會接收、顯示或儲存 API Key。

Railway Variables

請在 Crypto-Quant-Hub → Variables 新增：

BITGET_API_KEY=你的 Bitget API Key
BITGET_API_SECRET=你的 Bitget Secret
BITGET_API_PASSPHRASE=建立 API 時設定的 Passphrase
BITGET_NETWORK_MODE=testnet

先使用 testnet。本頁只提供 Demo Trading 下單，不開放主網真實下單。
